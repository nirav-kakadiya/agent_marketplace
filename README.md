## Adding a New Agent

1. Create `agents/my-agent/index.ts`
2. Extend `BaseAgent`, implement `handle(message)`
3. Import and register in `index.ts`

That's it. The bus routes everything. Zero changes to other agents.

## Key Principles

1. **Standard Interface** â€” every agent implements `handle(message): message`
2. **Message Bus** â€” agents don't talk directly, they send messages
3. **Composition** â€” agents can contain sub-agents (fractal architecture)
4. **Self-Tooling** â€” skill-builder creates integrations on the fly
